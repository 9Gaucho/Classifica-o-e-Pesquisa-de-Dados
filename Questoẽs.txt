4. O shellsort é uma ótima opção para arquivos de tamanho moderado e sua implementação é simples e requer uma quantidade de código pequena.
(a) Verdadeira
Justificativa: O Shellsort é considerado eficiente para tamanhos moderados de dados e possui uma implementação relativamente simples. Em comparação com algoritmos mais sofisticados, como o Quicksort e o Mergesort, o Shellsort pode ser mais fácil de entender e implementar.

5. No Quicksort, a pior escolha para pivô é:

(a) O extremo de uma lista já ordenada

Justificativa: Quando o Quicksort usa o extremo da lista como pivô em uma lista já ordenada (ou inversamente ordenada), ele pode degenerar em um desempenho quadrático O(n²), pois isso cria partições extremamente desbalanceadas.
6. O algoritmo Insertion Sort possui comportamento melhor do que o método da bolha (Bubblesort). Esta afirmação é:

(a) Verdadeira

Justificativa: O Insertion Sort geralmente possui um desempenho melhor do que o Bubble Sort, especialmente em listas parcialmente ordenadas. Enquanto o Bubble Sort sempre realiza n(n-1)/2 comparações, o Insertion Sort faz menos comparações e movimentações em listas que já estão parcialmente ordenadas.
7. O pivô recomendado para o método Quicksort é:

C) O elemento central

Justificativa: O uso do elemento central como pivô tende a reduzir a probabilidade de dividir a lista em partes extremamente desiguais, o que pode ocorrer se o primeiro ou o último elemento for escolhido, especialmente em listas já ordenadas ou quase ordenadas.
8. Suponha que você tenha a seguinte lista para realizar ordenação: [19, 1, 9, 7, 3, 10, 13, 15, 8, 12]. Qual opção representa a lista parcialmente ordenada após três passos completos de Bubble Sort?

e) [1, 3, 7, 9, 8, 10, 12, 13, 15, 19]

Justificativa: O Bubble Sort realiza múltiplas passagens pela lista, movendo os maiores elementos para o final da lista a cada passagem. Após três passagens completas, os maiores elementos (19, 15, 13) estarão no final da lista, mas o restante dos elementos ainda não estará completamente ordenado.
